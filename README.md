Steam-Games-Cartridges connects your digital Steam library to physical 2.5-inch SSD drives. This tool manages the link between a physical cartridge and your installed games. You plug in your drive, and the software recognizes your game files instantly. 

## 🛠 Prerequisites

Ensure your computer meets these requirements before you start:

* Windows 10 or Windows 11.
* A vacant USB 3.0 port for the cartridge dock.
* A 2.5-inch solid state drive formatted to NTFS. 
* A stable 5 megabits per second internet connection for the initial library sync.

## 🚀 Setting Up Your First Cartridge

This software links your physical drive to your Steam account.

1. Connect your 2.5-inch SSD cartridge to the dock.
2. Open the Steam-Games-Cartridges program from your desktop. 
3. The program detects the drive automatically. 
4. Select the "Initialize Cartridge" button within the app.
5. Name your cartridge to identify the game collection inside.
6. Select the games you want to sync to this specific drive. 
7. Wait for the file transfer process to finish.

## ⚙️ Using Your Hardware

Once configured, you switch between libraries by changing cartridges.

1. Exit the game you play. 
2. Safely remove the cartridge through the Windows taskbar icon if you need to.
3. Unplug the current drive.
4. Plug in a different cartridge.
5. Watch the software update the Steam library view. 
6. Play your games directly from the external drive.

## 💡 Managing Storage

The software handles file paths so Steam knows where your games live. You do not need to move files manually or change Steam settings. When you plug in a drive, the program writes the correct location data to your system. When you unplug the drive, it removes those shortcuts to keep your computer tidy.

## 🛠 Troubleshooting

If the software does not see your drive, try these steps:

* Check the physical connection. USB 3.0 cables often work better than older USB 2.0 cables.
* Ensure your drive appears in "This PC" under File Explorer.
* Restart the Steam-Games-Cartridges application.
* Check if the drive uses the NTFS format. The software cannot read drives formatted for Apple computers or gaming consoles.
* Disable antivirus software temporarily to see if it blocks the drive recognition.
* Update your Windows USB drivers through the Device Manager.
